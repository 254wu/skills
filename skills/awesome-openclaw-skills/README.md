## 🚀 安装技能

### 方式 1：ClawHub CLI
```bash
clawhub install <skill-slug>
```

### 方式 2：手动安装
```bash
# 复制技能文件夹到
~/.openclaw/workspace/skills/
```

### 方式 3：直接从本仓库
```bash
git clone https://github.com/254wu/skills.git
# 复制需要的技能文件夹到 ~/.openclaw/workspace/skills/
```
